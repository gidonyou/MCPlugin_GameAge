# 패치 내역
모든 패치 기록은 여기에 첨부 될것입니다. 버전마다 도움을 주실경우 패치에 추가해 드립니다.
*패치에 노트에 없는 추가 내역이 있을수 있습니다. 찾으실 경우 저에게 알려주시면 추가하겠습니다*

### 베타 테스트 후

#### 패치
  - 데스노트 메세지에 '우클릭' 이라는것을 명시합니다.
  - 상대를 잘못공격했을시 화면에 경고메세지와 소리가 나옵니다.
  - 1:1 상황에는 1등이 꼴등 공격 추가 데미지가 없습니다. (미테스트)
  - `ENDER_PEARL` (엔더진주) 다음 업데이트에서 삭제.
  - `LEGACY` (순정 꼬리잡기)를 어떻게든 고칠려고 했지만 실패만 계속된 관계로 다음기회에...
  
#### 알려진 버그
  - 게임 끝난후 가끔 메세지가 3번뜹니다.

### 게임 시작/끝
  - 타이틀: 시작할때 카운트 다운을 추가하였습니다.  
  시작 카운트 다운은 `COUNTDOWN_TIME` 에서 변경 가능.
  - 타이틀: 게임이 끝났을때 우승자를 알려줍니다.
  - Game End --> Dead player teleport to last player

### 게임중
  - 타이틀: 아웃된 플레이어를 화면에 알려줍니다.

### 커멘드
  - `/play cu` 추가 - 플러그인 버전과 업데이트를 확인하실수 있습니다.

### 설정
  - 설정 추가: `IS_INVISIBILITY`  
    * 설명: 게임 아이템 '투명화 시계' 지속시간을 설정합니다.  
    * 방법: W 0~ (초)
    * 기본값: 10 - 10초
  - 설정 추가: `COUNTDOWN_TIME`:  
    * 설명: 게임 시작할때 카운트 다운 시간을 정합니다.  
    * 방법: W 0~ (초)
    * 기본값: 5 - 5초
  - 설정 추가: `LEGACY`:
    * 설명: 꼬리잡기를 오리지널하게 진행합니다.
    * 방법: B 0 - 일반  |  1 - 오리지널
    * 기본값: 0 - 일반
  - 설정 추가: `ADD_ENCHANTMENT`:
    * 설명: 아이템에 인첸트를 추가합니다 .
    * 방법: B 0 - 추가안함  |  1 - 추가
    * 기본값: 0 - 추가안함
    
  기본값 설정 변경:  
    - `MINY` : 4 --> 20  
    - `MAXY` : 64 --> 180
    - `ALLOW_BREAK_SPLATE` : 1 --> 0
    - `AUTO_BORDER` : 3 --> 0
